# Statistical Process Control (SPC) - Quality Control App

A mobile application built with **Python**, **Kivy**, and **Matplotlib** designed for Quality Engineers to perform Statistical Process Control (SPC) analyses directly on mobile devices.

## Features
* **Multiple Control Charts Support:**
  * Xbar - R Chart (Variables, Subgroup > 1)
  * I - MR Chart (Individual Values)
  * p Chart (Fraction Defective)
  * c Chart (Number of Defects)
* **Dynamic Data Grid:** Automatically generates input fields based on sample size and rows.
* **Interactive Charting:** Highlights Out-of-Control (OOC) points in **RED**.
* **Automatic Recalculation:** Option to delete out-of-control samples and recalculate UCL, LCL, and CL limits dynamically.
